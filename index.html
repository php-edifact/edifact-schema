Schema Structure

  • Data Elements: Simple fields with type, maxLength, and optional code lists
  • Composite Data Elements: Objects containing multiple data elements
  • Segments: Objects containing data elements and composite data elements
  • Messages: Complete message schemas with segments and segment groups

Example Usage

JSON Schema (any language)

{
  "$schema": "https://json-schema.org/draft/2020-12/schema",
  "$ref": "https://php-edifact.github.io/edifact-schema/D95B/segment/ADR.edifact.schema.json"
}

Python

import json
import jsonschema

# Load schema from URL
import urllib.request
with urllib.request.urlopen('https://php-edifact.github.io/edifact-schema/D95B/message/APERAK.edifact.schema.json') as url:
    schema = json.loads(url.read().decode())

# Or load locally
with open('D95B/message/APERAK.edifact.schema.json') as f:
    schema = json.load(f)

message = {
    "UNH": {...},
    "BGM": {...},
    "UNT": {...}
}

jsonschema.validate(message, schema)

JavaScript

const Ajv = require('ajv');
const fetch = require('node-fetch');

const url = 'https://php-edifact.github.io/edifact-schema/D95B/message/APERAK.edifact.schema.json';
const response = await fetch(url);
const schema = await response.json();

const ajv = new Ajv({ schemaId: 'auto' });
ajv.addSchema(schema);

const message = { /* your message */ };
const valid = ajv.validate(url, message);

JSON Schema Viewer

JSON Schema files are raw JSON. To visualize and explore the schema structure, you can use online tools:

  • JSON Schema Validator - Paste schema and validate
  • JSONSchema.io - Interactive schema builder and viewer
  • QuickType Schema Viewer - Visualize schema structure

Tip: For VS Code users, install the "JSON Schema Validation" extension for inline validation.

Regenerating Schemas

cd edifact-schema
composer install
php schema.php          # Generate D95B (default)
php schema.php D96A    # Generate specific version
php schema.php all     # Generate all versions
